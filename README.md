# TELCO.PREMIUM

Premium phone brand landing page built with Next.js, TypeScript, and plain CSS. Features responsive layouts, animated navigation, product showcase, and support section.

## Pages

- **/** — Home / Phones landing page with hero, specs grid, product gallery, and CTA
- **/accessories** — Accessories catalog with filterable product grid and tech showcase
- **/support** — Support hub with search, help categories, FAQ accordion, and contact section

## Tech Stack

- Next.js (Pages Router)
- TypeScript
- Plain CSS with CSS Modules
- Google Fonts (Montserrat, Inter)
- Material Symbols

## Getting Started

```bash
npm run dev
```
